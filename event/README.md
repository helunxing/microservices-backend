# Event Microservice

This microservice return events data and handle change request.

Default port number can find in [here](../README.md#local-urls).

## GET `/events`

Get all events.

## POST `/event`

Create a new event.

## GET `/event/{eventId}`

Get a event by id.

## PUT `/event/{eventId}`

Update a event by id.

## DELETE `/event/{eventId}`

Delete a event by id.
